# Data-Analytics-Capstone

Introduction

Major League Baseball has a rich set of statistics that can be used for many different reasons. One specific, electrifying metric is the number of home runs a player hits in a season. As a baseball fan since the Sosa/McGwire days, knowing if and when the next exciting home run race will happen has always been of interest to me. The question I am going to try to answer is - what statistics are most likely to predict the number of home runs a player has in a season.
